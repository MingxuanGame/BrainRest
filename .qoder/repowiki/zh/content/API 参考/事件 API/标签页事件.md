## 简介

标签页事件描述标签页的创建、关闭、URL 变更与激活，均继承自 `TabEvent`（含 `tabId`）。

## TabEvent 基类

```ts
export interface TabEvent extends Event {
  tabId: number;
}
```

## 字段契约

| 事件           | `type`          | 额外字段              | 类型     |
|----------------|-----------------|-----------------------|----------|
| `TabCreated`   | `tab_created`   | —                     | —        |
| `TabClosed`    | `tab_closed`    | —                     | —        |
| `TabChanged`   | `tab_changed`   | `old_url` / `new_url` | `string` |
| `TabActivated` | `tab_activated` | `windowId`            | `number` |

## 产生来源

[src/background/TabListener.ts](file://src/background/TabListener.ts) 监听 `chrome.tabs` 的 `onCreated` / `onRemoved` /
`onUpdated`（仅在 `changeInfo.url` 存在时）/ `onActivated`，构造上述事件并 `queue.push`。它们是疲劳指标 T（标签页切换）的来源，由
`TabSwitchAnalyzer` 消费。
