## 简介

窗口相关事件包含窗口焦点变化 `WindowFocus` 与全屏切换 `FullscreenChange`，二者直接继承 `Event`。它们参与休息权重 `R` 的判定。

## WindowFocus

```ts
export interface WindowFocus extends Event {
  type: 'focus' | 'blur';
  windowId: number;
}
```

由 [src/background/WindowFocusListener.ts](file://src/background/WindowFocusListener.ts) 产生：
`chrome.windows.onFocusChanged` 中，窗口 ID 为 `WINDOW_ID_NONE` 时为 `blur`，否则为 `focus`，`url` 为空串。

## FullscreenChange

```ts
export interface FullscreenChange extends Event {
  type: 'fullscreen_change';
  active: boolean;
}
```

由 [src/content/DomListener.ts](file://src/content/DomListener.ts) 监听 `fullscreenchange` 产生，`active` 表示是否进入全屏。

## 与休息权重的关系

| 事件条件                                | 休息权重贡献           |
|-----------------------------------------|------------------------|
| 窗口失焦持续 ≥ `WINDOW_BLUR_MS = 30000` | `windowBlur = 50`      |
| 视频全屏（`setVideoFullscreen(true)`）  | `videoFullscreen = 30` |
