## 简介
用户界面事件描述页面内交互，均继承自 `UiEvent`（含可选目标元素信息 `targetTag` / `targetId` / `targetClass`）。

## UiEvent 基类
```ts
export interface UiEvent extends Event {
  targetTag?: string;
  targetId?: string;
  targetClass?: string;
}
```

## 字段契约
| 事件 | `type` | 额外字段 |
|------|--------|----------|
| `UiClickEvent` | `click` | `button`, `clientX`, `clientY` |
| `UiKeyEvent` | `keydown` \| `keyup` | `key`, `code`, `altKey`, `ctrlKey`, `shiftKey`, `metaKey` |
| `UiMouseMoveEvent` | `mousemove` | `clientX`, `clientY` |
| `UiScrollEvent` | `scroll` | `scrollX`, `scrollY` |
| `UiTouchEvent` | `touchstart` \| `touchend` \| `touchmove` | `identifier`, `clientX`, `clientY`, `force`, `radiusX`, `radiusY`, `rotationAngle` |

## 产生来源
[src/content/DomListener.ts](file://src/content/DomListener.ts) 采集：`mousemove`（以 `MOUSE_MOVE_SAMPLE_MS = 200` 采样）、`click`、`keydown`/`keyup`、`scroll`、`touch*`（遍历 `changedTouches`）。这些事件是鼠标方向熵（E）、眼手延迟（D）与事件频率（I）指标的输入。
