## 简介
媒体事件 `MediaEvent` 用于描述页面媒体（如视频）的播放/暂停状态，直接继承自 `Event`。

## MediaEvent 字段契约
```ts
export interface MediaEvent extends Event {
  type: 'media';
  tab_id: number;
  paused: boolean;
  url: string;
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `type` | `'media'` | 固定标识 |
| `tab_id` | `number` | 所属标签页 |
| `paused` | `boolean` | 是否暂停 |
| `url` | `string` | 媒体所在页面 URL |

## 实现状态说明
> [!NOTE]
> `MediaEvent` 仅有模型定义，当前代码库中**没有任何监听器或模块构造/发射该事件**，也没有分析器消费它。本文如实标注其为"已定义、未接入"，以免误导读者认为存在媒体采集链路。视频相关的休息判定当前是通过 `FullscreenChange` → `setVideoFullscreen` 实现的，而非 `MediaEvent`。
