## 简介
用户界面事件描述页面内的用户交互（鼠标、键盘、滚动、触摸），均继承自 `UiEvent`，由内容脚本 `DomListener` 采集并经 Port 上报。它们驱动疲劳指数中的鼠标熵、眼手延迟、事件频率等指标。

## UiEvent 基接口
```ts
export interface UiEvent extends Event {
  type: string;
  targetTag?: string;    // 目标元素标签名
  targetId?: string;     // 目标元素 id
  targetClass?: string;  // 目标元素 class
}
```
三个 target 字段均为可选，用于标识交互目标。

## 采集来源
`DomListener` 在 `document` 上注册监听器：
- `mousemove`：以 `MOUSE_MOVE_SAMPLE_MS = 200` 采样，避免高频事件淹没频率统计。
- `click`、`keydown`、`keyup`、`scroll`。
- `touchstart`/`touchend`/`touchmove`：对 `changedTouches` 中每个触点分别发一条事件。

每条事件都记录 `url = window.location.href` 与 target 元素信息。
