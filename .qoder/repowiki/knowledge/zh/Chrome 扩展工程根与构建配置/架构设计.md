本模块为纯配置层，不承载业务代码，职责是定义构建、类型与校验规则：
- 构建入口：vite.config.ts 启用 @vitejs/plugin-react 与 @crxjs/vite-plugin，从 src/manifest.ts 读取 Manifest V3 描述，并将 src/content/index.ts 注册为独立 content script；构建产物关闭压缩并开启 sourcemap。
- 运行时入口：index.html 挂载 <div id="root"> 并通过 ES module 加载 /src/main.tsx。
- 类型系统采用 Project References 拆分：tsconfig.json 聚合 tsconfig.app.json（浏览器端，target ES2023 + DOM，types 包含 vite/client 与 chrome）和 tsconfig.node.json（Node 端，仅编译 vite.config.ts），两者均使用 bundler/nodenext 解析、verbatimModuleSyntax、moduleDetection: force 以及 erasableSyntaxOnly 等现代 TS 选项。
- 代码质量：ESLint Flat Config (eslint.config.js) 组合 @eslint/js、typescript-eslint、react-hooks、react-refresh，全局暴露 browser 环境；忽略 dist 目录。
- 脚本命令：dev/preview 走 Vite，build 先执行 tsc -b 再 vite build，debug 额外生成 sourcemap。
